# Basic Java Syntax: A Core Overview\n\nJava is an object-oriented, class-based, concurrent, secure and platform-independent programming language. Understanding its basic syntax is the first step to becoming a Java developer.\n\n## 1. Java Program Structure\n\nEvery Java application starts execution from the `main` method. A typical basic Java program looks like this:\n\n```java\npublic class MyFirstJavaProgram {\n    public static void main(String[] args) {\n        // Your code goes here\n        System.out.println(\"Hello, Java!\");\n    }\n}\n```\n\n*   `public class MyFirstJavaProgram`: Defines a class named `MyFirstJavaProgram`. In Java, all code resides within classes. `public` means it's accessible from anywhere.\n*   `public static void main(String[] args)`: This is the entry point of any Java application.\n    *   `public`: Access modifier, visible to all.\n    *   `static`: Allows the `main` method to be called without creating an object of the class.\n    *   `void`: Indicates that the method does not return any value.\n    *   `main`: The name of the method. It's a special keyword for the JVM.\n    *   `(String[] args)`: Command-line arguments passed as an array of strings.\n*   `System.out.println(\"Hello, Java!\");`: Prints the string \"Hello, Java!\" to the console, followed by a new line.\n\n## 2. Variables and Data Types\n\nVariables are containers for storing data values. In Java, every variable must be declared with a data type.\n\n### Primitive Data Types:\n\n| Type    | Size    | Range                                    |\n| :------ | :------ | :--------------------------------------- |\n| `byte`  | 1 byte  | -128 to 127                              |\n| `short` | 2 bytes | -32,768 to 32,767                        |\n| `int`   | 4 bytes | -2,147,483,648 to 2,147,483,647          |\n| `long`  | 8 bytes | Very large range                         |\n| `float` | 4 bytes | Single-precision floating-point          |\n| `double`| 8 bytes | Double-precision floating-point          |\n| `boolean`| 1 bit   | `true` or `false`                        |\n| `char`  | 2 bytes | Single Unicode character ('a', '1', '$') |\n\n### Variable Declaration and Initialization:\n\n```java\nint age = 30;\ndouble price = 19.99;\nboolean isActive = true;\nchar initial = 'J';\nString name = \"Java\"; // String is a non-primitive (object) type\n```\n\n## 3. Operators\n\nOperators are used to perform operations on variables and values.\n\n*   **Arithmetic:** `+`, `-`, `*`, `/`, `%` (modulus)\n*   **Assignment:** `=`, `+=`, `-=`, `*=`, `/=`, `%=`\n*   **Comparison:** `==` (equal to), `!=` (not equal), `>` (greater than), `<` (less than), `>=` (greater than or equal to), `<=` (less than or equal to)\n*   **Logical:** `&&` (AND), `||` (OR), `!` (NOT)\n*   **Unary:** `++` (increment), `--` (decrement)\n\n### Example:\n\n```java\nint x = 10;\nint y = 5;\nint sum = x + y; // sum is 15\nboolean isEqual = (x == y); // isEqual is false\n```\n\n## 4. Control Flow Statements\n\nControl flow statements determine the order in which statements are executed.\n\n### If-Else Statements:\n\n```java\nint score = 85;\nif (score >= 90) {\n    System.out.println(\"Grade A\");\n} else if (score >= 80) {\n    System.out.println(\"Grade B\");\n} else {\n    System.out.println(\"Grade C\");\n}\n```\n\n### Switch Statements:\n\n```java\nint day = 3;\nswitch (day) {\n    case 1:\n        System.out.println(\"Monday\");\n        break;\n    case 2:\n        System.out.println(\"Tuesday\");\n        break;\n    default:\n        System.out.println(\"Another day\");\n}\n```\n\n### Loops (for, while, do-while, for-each):\n\n**For Loop:**\n\n```java\nfor (int i = 0; i < 5; i++) {\n    System.out.println(i);\n}\n```\n\n**While Loop:**\n\n```java\nint i = 0;\nwhile (i < 5) {\n    System.out.println(i);\n    i++;\n}\n```\n\n**Do-While Loop:**\n\n```java\nint i = 0;\ndo {\n    System.out.println(i);\n    i++;\n} while (i < 5);\n```\n\n**For-Each Loop (Enhanced For Loop):**\n\n```java\nString[] names = {\"Alice\", \"Bob\", \"Charlie\"};\nfor (String name : names) {\n    System.out.println(name);\n}\n```\n\n## 5. Comments\n\nComments are used to explain code and make it more readable. They are ignored by the compiler.\n\n*   **Single-line comment:** `// This is a single-line comment`\n*   **Multi-line comment:**\n\n    ```java\n    /*\n     * This is a multi-line comment.\n     * It can span multiple lines.\n     */\n    ```\n*   **Documentation comment:** (Used for Javadoc)\n\n    ```java\n    /**\n     * This is a documentation comment.\n     * It can be used to generate API documentation.\n     */\n    ```\n\nThis overview covers the very basics of Java syntax, providing a foundation for writing simple Java programs. As you progress, you'll delve deeper into object-oriented concepts, advanced data structures, and more.
